To use this product, follow these 5 steps:
    1. Go to the Github repository of our project: https://github.com/julie-annej/McHacks8
    2. Download and extract the file on your computer
    3. Open Google Chrome and go to Chrome extension : chrome://extensions
    4. Activate Developper mode and click on Load Unpacked
